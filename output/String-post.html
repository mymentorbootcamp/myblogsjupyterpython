# Membuat set dari nilai ASCII karakter-karakter dalam string my_string = "Hello" ascii_set = {ord(c) for c in my_string} print(ascii_set) Output: {72, 101, 108, 111} Dalam contoh di atas, ord(c) mengembalikan nilai ASCII dari setiap karakter c dalam string "Hello". Kemudian, operator {} digunakan untuk membuat set yang berisi nilai-nilai tersebut. Set hanya akan menyimpan nilai-nilai unik, sehingga setelah dijalankan, hanya ada satu nilai 108 yang mewakili karakter 'l'.

________ Slicing of a string ________my_string = "Hello, World!" # Mengambil potongan string dari indeks 0 hingga 4 (tidak inklusif) substring = my_string[0:5] print(substring) # Output: Hello # Mengambil potongan string dari indeks 7 hingga akhir substring2 = my_string[7:] print(substring2) # Output: World! # Mengambil potongan string dari awal hingga indeks 5 (tidak inklusif) substring3 = my_string[:5] print(substring3) # Output: Hello Dalam contoh di atas, perlu diingat bahwa rentang indeks pada potongan string adalah [start:stop), yang berarti karakter pada indeks "stop" tidak termasuk dalam hasilnya.

# format() method
"""
The format() method formats the specified value(s) and insert them inside the string's placeholder. 
The placeholder is defined using curly brackets: {}.
"""
txt = "Hello {word}"
print(txt.format(word = 'World!'))
message1 = 'Hi, My name is {} and I am {} years old.'
print(message1.format('Bob', 36))
message2 = 'Hi, My name is {name} and I am {number} years old.'
print(message2.format(name ='Bob', number = 36))
message3 = 'Hi, My name is {0} and I am {1} years old.'
print(message3.format('Bob', 36))
