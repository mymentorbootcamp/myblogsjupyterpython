03_Lists, Tuples, and Sets

Monday, January 15, 2024 on Python

Lists, Tuples, and Sets¶

1. List (Daftar): my_list = [1, 2, 3, 'a', 'b', 'c'] Sifat Utama: Mutable: Isi list dapat diubah setelah pembuatan. Dapat berisi elemen dengan tipe data yang berbeda. 2. Tuple: my_tuple = (1, 2, 3, 'a', 'b', 'c') Sifat Utama: Immutable: Setelah dibuat, elemen tuple tidak dapat diubah. Dapat berisi elemen dengan tipe data yang berbeda. 3. Set: my_set = {1, 2, 3, 'a', 'b', 'c'} Sifat Utama: Unordered: Tidak ada urutan tertentu dalam set. Mutable: Isi set dapat diubah (elemen dapat ditambah atau dihapus). Hanya dapat berisi elemen unik, tidak ada duplikasi. Perlu diingat bahwa List menggunakan tanda kurung siku [ ], Tuple menggunakan tanda kurung ( ), dan Set menggunakan tanda kurung kurawal { }.

Tuples¶

Tuples are immutable lists and cannot be changed in any way once it is created. - Tuples are defined in the same way as lists. - They are enclosed within parenthesis and not within square braces. - Tuples are ordered, indexed collections of data. - Similar to string indices, the first value in the tuple will have the index [0], the second value [1] - Negative indices are counted from the end of the tuple, just like lists. - Tuple also has the same structure where commas separate the values. - Tuples can store duplicate values. - Tuples allow to store several data items including string, integer, float in one variable

SETS¶

- Set is one of 4 built-in data types in Python used to store collecti ons of data including List, Tuple, and Dictionary - Sets are unordered, but you can remove items and add new items. - Set elements are unique. Duplicate elements are not allowed. - A set itself may be modified, but the elements contained in the set must be of an immutable type. - Sets are used to store multiple items in a single variable. - You can denote a set with a pair of curly brackets {}.
In [ ]:
